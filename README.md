# 📅 ContentAI – AI Content Calendar Generator

> Generate 30 days of social media content in seconds using AI.  
> Built by **vibetechbee** · Powered by Claude AI

---

## ✨ What It Does

Paste your brand name, pick your niche and platform, and get a full content calendar with:

- **Topic ideas** for every day
- **Attention-grabbing hooks**
- **Full captions** with emojis
- **Hashtag sets**
- **Call-to-action** lines
- One-click copy per post

---

## 🚀 Run It Locally

### Step 1 – Clone this repo

```bash
git clone https://github.com/YOUR_USERNAME/ai-content-calendar.git
cd ai-content-calendar
```

### Step 2 – Install dependencies

```bash
npm install
```

### Step 3 – Run the app

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
